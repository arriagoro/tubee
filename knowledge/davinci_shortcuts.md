# DaVinci Resolve Keyboard Shortcuts — Complete Reference
### Organized by Category | Mac shortcuts shown with ⌘ (Cmd), ⌥ (Opt), ⇧ (Shift), ^ (Ctrl)
### Windows/Linux: Replace ⌘ with Ctrl, ⌥ with Alt

## Navigation & Playback

| Shortcut | Action |
|----------|--------|
| Space | Play / Stop |
| L | Play forward (tap multiple times: 2×, 4×, 8×, 16×) |
| J | Play reverse (tap multiple times for faster) |
| K | Stop playback |
| K + L | Play forward frame-by-frame (hold K, tap L) |
| K + J | Play reverse frame-by-frame (hold K, tap J) |
| ← / → | Step one frame back / forward |
| ⇧ + ← / → | Move playhead 1 second back / forward |
| ↑ / ↓ | Jump to previous / next edit point |
| Home | Go to start of timeline |
| End | Go to end of timeline |
| ; / ' | Move playhead to previous / next marker |
| / | Play around current position (with pre/post roll) |
| ⌥ + / | Play In to Out range |

## Customization

**To customize any shortcut:**
1. DaVinci Resolve → Keyboard Customization (⌘ + ⌥ + K)
2. Search for the command
3. Click the existing shortcut → press your desired key combo
4. Save as a custom preset

**Popular Custom Shortcuts:**
- Map "Blade" to `C` (Premiere Pro habit)
- Map "Ripple Delete" to `⇧ + Delete`
- Map "Full Screen Preview" to `~` (tilde)
- Map "Swap Clips" to `⌘ + ⇧ + S`
